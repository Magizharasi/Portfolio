Malware Analysis using Sysinternals Tools

Academic Research Project | Windows OS

Analyzed Agent Tesla keylogger malware in a controlled virtual environment using Process Explorer, Process Monitor, Autoruns, and Wireshark. Detected process injection, registry-based persistence, and C2 communication via encrypted network traffic. Observed malware impact on system performance and proposed enhancements using automated sandboxing and ML for anomaly detection.

Windows OS Wireshark Process Explorer Process Monitor Autoruns Virtual Environment Network Analysis

Zero-Day Attack Detection in IoT Devices

Samsung SNH-1011-N Webcam Security Analysis

Developed an unsupervised deep learning model (Autoencoder) to detect zero-day vulnerabilities in IoT network traffic. Applied t-SNE for latent space visualization and Logistic Regression for anomaly classification. Processed real-world IoT traffic data including Gafgyt botnet attacks, achieving ~48% accuracy with scope for optimization using VAE and Transformer-based models.
